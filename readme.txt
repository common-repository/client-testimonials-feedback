Use this plugin to get client testimonial feedback slider and listing in your wordpress.

== Description ==
This plugin can add functionality of client testimonial feedback slider and client testimonial feedback listing in your WordPress website. You can show client testimonial feedback on any page in listing view OR in slider.

<strong>Features</strong>

* Testimonial select type slider OR listing options in testimonial settings admin
* Testimonial choose background and text color customization options in testimonial settings admin
* Shortcode System.
* Use this shortcode [client-testimonial-feedback] show the testimonial.
* Wordpress Testimonial Custom Post Enabled.
* Easy to Use.
* You can use it in posts or pages.

== Installation ==
Steps to follow.
1. Click on \"Add New\" button in \"Plugins\" menu.
2. Search for \"Client Testimonials Feedback\" plugin.
3. Click on the \"Install\" button & then after \"Active\".
4. Use following shortcode on WordPress page: [client-testimonial-feedback] OR if you want to add it into php template then use it as do_shortcode('[client-testimonial-feedback]')
5. That's it. You\'re ready to use the plugin.
